# SAFE PUBLIC CLAIMS — YoutubeFocus

---

## SAFE TO CLAIM

These are accurate, verifiable statements about the product:

1. **Hides YouTube Shorts, the home feed, related videos, comments, Trending, Explore, and notification distractions** — all implemented via DOM manipulation in the content script
2. **One-click Focus Mode toggle** — single click in the popup enables or disables the distraction-free state
3. **Timed Lock supports 25, 50, 90, and custom-minute sessions** — all four session durations are implemented
4. **Light and dark popup themes** — both themes are implemented and user-selectable
5. **No analytics, no remote code, no data collection** — the extension makes no network requests; all logic is local
6. **Preferences stored locally via Chrome's local storage API** — no external storage or account required
7. **Built with Chrome Manifest V3** — uses the current MV3 standard required by Chrome Web Store
8. **Chrome Web Store ready** — structured as a valid MV3 extension with manifest, icons, and content scripts

---

## DO NOT CLAIM

Avoid these — they are misleading, unverifiable, or likely to set wrong expectations:

1. **"Blocks all distractions on YouTube"** — it hides specific UI elements; a determined YouTube update could break any of them temporarily; "hides" is more accurate than "blocks"
2. **"Boosts productivity by X%"** — any percentage claim on focus/productivity is fabricated without a study to back it up
3. **"Works on all browsers"** — it's a Chrome extension; it may or may not work in Chromium-based browsers (Edge, Brave) and does not work in Firefox or Safari without separate ports
4. **"YouTube can't detect it"** — make no guarantees about YouTube's ability to detect or circumvent content script behavior
5. **"Permanently removes distractions"** — YouTube updates its DOM regularly; features may require periodic maintenance to keep working against YouTube's changes
